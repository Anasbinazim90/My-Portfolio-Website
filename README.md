### 🎨 Modern Design

- **Glassmorphism Effects** - Modern glass-like UI components
- **Smooth Animations** - Custom keyframe animations throughout

## 🔧 Customization

### Update Personal Information

Edit `index.html`:

- Change name, title, and description
- Update WhatsApp numbers
- Modify email addresses
- Add/remove social links

### Change Colors

Edit CSS variables in `style.css`:

```css
:root {
  --primary-color: #ff00ee;
  --secondary-color: #00f5ff;
  --dark-bg: #0f0a0a;
}
```

### Add Your Images

Replace image sources:

- Profile image: `MdSay.png`
- Or use the built-in SVG fallbacks
